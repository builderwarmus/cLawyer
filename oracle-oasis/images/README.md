# Images for the Oasis Aqualounge proposal site

Drop image files into this folder using the **exact filenames** below. The site
loads each one automatically; any file that's missing simply falls back to the
current gradient placeholder, so you can add them one at a time.

All slots are optional — the site looks complete without them, and upgrades as
you add photos.

| Filename | Where it appears | Recommended size | Notes |
|---|---|---|---|
| `hero.jpg` | Full-bleed background behind the opening headline | 2400 × 1400 (landscape) | Sits under a dark overlay, so a moody/wide shot works best. Keep detail toward the right/center. |
| `floor-lounge.jpg` | "The descent · four floors" — tile 1 | 900 × 1200 (portrait) | The lounge / entry level. |
| `floor-waters.jpg` | Four-floor tile 2 | 900 × 1200 (portrait) | Pool, courtyard, sauna. |
| `floor-playrooms.jpg` | Four-floor tile 3 | 900 × 1200 (portrait) | Playroom level (tasteful/ambient). |
| `floor-quarters.jpg` | Four-floor tile 4 | 900 × 1200 (portrait) | Private quarters. |
| `og-cover.jpg` | Social share preview (Facebook, iMessage, etc.) | 1200 × 630 (landscape) | Referenced by the Open Graph / Twitter meta tags. |

## Guidance
- **No guests.** Per the club's policy, use interiors, ambience, bar, pool and
  sauna only (or fully-consented talent). This mirrors what the proposal itself
  recommends.
- **Format:** `.jpg` for photos (smaller); `.png` only if you need transparency.
  If you send `.webp` or a different extension, tell me and I'll update the
  filenames in `index.html`.
- **Optimize:** aim for < ~400 KB each so the page stays fast. I can compress
  them for you if you send full-res originals.

## How to send them
Just upload the images to our chat and I'll place, optimize and wire them in —
both into this repo and into the live-preview artifact (embedded so the preview
shows them too).
